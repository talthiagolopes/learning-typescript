# Learning TypeScript
Learning TypeScript \o/

Install typescript

` npm i typescript -g `

Watch mode
` tsc *.ts --watch `

or a especific file
` tsc my-file.ts --watch `

Create tsconfig
` tsc --init `

build with tsconfig
` tsc --build tsconfig.json `
